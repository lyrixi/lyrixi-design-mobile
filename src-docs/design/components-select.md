# 选择组件

## 通用属性

| Name       | Description  | Data                  | Default |
| ---------- | ------------ | --------------------- | ------- |
| value      | 值           | [{id:'', name: ''}]   | Empty   |
| onChange   | 修改         | ([{id:'', name: ''}]) | Empty   |
| multiple   | 多选         | true/false            | false   |
| checkable  | 是否可选     | true /false           | true    |
| allowClear | 是否允许清空 | true/false            | false   |
