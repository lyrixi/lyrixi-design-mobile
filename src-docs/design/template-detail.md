# 详情页

详情页向用户展示一个对象的完整信息，主要用于信息浏览，允许对该对象发起编辑等操作。

<img class="preview-img no-padding" src="./assets/images/template-detail/normal.png" />

## 设计目标

提高信息浏览和搜寻效率，便捷执行操作。
