# 加载进度

在操作需要一段时间（一般为超过 2 秒）完成时系统应即时给予提醒，明确告知加载的状态或加载进度条，保持与用户的沟通。｜注：若加载时间较长，应提供取消操作。

当用户不必等待较长时间的加载时使用。
