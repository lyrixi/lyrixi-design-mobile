# 栅格

栅格可以有效的保证页面的一致性、逻辑性、加强团队协作和统一。

### 栅格构成

· 页面宽度W ：指页面的的整体宽度，由W代表。

· 栅格个数A ：指整体页面有多少个栅格。

· 水槽个数B：指栅格之间的间距有多少个。

· 栅格宽度C：指每一个栅格的宽度（Column）

· 水槽宽度G ：指每个水槽的宽度（Gutter）

· 边距：指第一个栅格距离页面边框的距离（Margin）

### 栅格类型

· 12等分：整体页面由12个栅格等分，适用于业务信息分组较少的中后台页面设计

· 24等分 ：整体页面由24个栅格等分，适用于业务信息量大，场景复杂的页面设计
