# DatePicker

日期选择器组件，提供多种日期选择方式。

## 何时使用

- 需要选择日期时
- 需要选择时间时
- 需要选择日期范围时
- 需要选择周/月/年时

## DatePicker.Combo

### 属性

| 属性               | 说明       | 类型                                                                         | 默认值   |
| ------------------ | ---------- | ---------------------------------------------------------------------------- | -------- |
| modalProps         | 模态框属性 | `object`                                                                     | -        |
| defaultPickerValue | 默认选择值 | `Date`                                                                       | -        |
| onError            | 错误回调   | `(error: Error) => void`                                                     | -        |
| value              | 当前值     | `Date`                                                                       | -        |
| type               | 选择类型   | `'year' \| 'quarter' \| 'month' \| 'date' \| 'time' \| 'datetime' \| 'week'` | `'date'` |
| min                | 最小日期   | `Date`                                                                       | -        |
| max                | 最大日期   | `Date`                                                                       | -        |
| hourStep           | 小时步长   | `number`                                                                     | -        |
| minuteStep         | 分钟步长   | `number`                                                                     | -        |
