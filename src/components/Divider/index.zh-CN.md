# Divider

分割线组件，用于分隔内容。

## 何时使用

- 需要分隔不同内容区域时
- 需要视觉上的内容分组时
- 需要水平分割线时

## 示例

<code src="./demos/demo1.jsx"></code>

## Divider

### 属性

| 属性     | 说明       | 类型        | 默认值 |
| -------- | ---------- | ----------- | ------ |
| children | 分割线内容 | `ReactNode` | -      |

### Ref

| 属性       | 说明       | 类型                   |
| ---------- | ---------- | ---------------------- |
| rootDOM    | 根元素     | `HtmlDivElement`       |
| getRootDOM | 获取根元素 | () => `HtmlDivElement` |
