# Share

分享组件，用于内容分享功能。

## 何时使用

- 需要分享内容时
- 需要社交分享时
- 需要多平台分享时
- 需要自定义分享时

## Share.Main

### 属性

| 属性      | 说明       | 类型     | 默认值 |
| --------- | ---------- | -------- | ------ |
| className | 自定义类名 | `string` | -      |
| style     | 自定义样式 | `object` | -      |
| shareTo   | 分享配置   | `object` | -      |

### Ref

| 属性       | 说明             | 类型                           |
| ---------- | ---------------- | ------------------------------ |
| mainDOM    | 主容器元素       | `HtmlDivElement`               |
| getMainDOM | 获取主容器元素   | () => `HtmlDivElement`         |
| support    | 检查是否支持分享 | `(shareTo: object) => boolean` |
